Why This is Exploitable:

  1. Server Vulnerability: CORS misconfiguration exists at server level (access-control-allow-origin: *)
  2. Token Exposure: /api/auth/session returns JWT tokens with valid session cookies
  3. XSS Enables Automation: Common web vulnerability enables automatic token theft
  4. Impact: Sensitive user data accessible via cross-origin requests
